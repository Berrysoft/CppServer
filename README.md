# Linux中的C++大作业
## 使用说明
默认已经编译好程序。可以在目录下运行以下命令编译并启动服务器：
```
make run
```
make命令需要保证`obj`与`bin`文件夹存在。

然后在浏览器中输入`http://127.0.0.1:3342/`访问。
## 功能说明
在服务器端输入`r`刷新模块加载，输入`c`清理超时连接，输入`q`退出。

默认提供了以下模块：
* 查看文件
* CPU与进程信息
* 系统版本、时间与内存信息
* 硬盘信息
* 说明文档（简单Markdown解释器）

查看文件模块是必需的，如果删除会导致400错误。

说明文档用的是“../README.md”绝对路径。

其实系统信息也可以作为文件打开。
## 程序文件
本程序需要以下文件才能相应HTTP请求：
* server

本程序需要以下文件才能正常使用：
* modules
* file.so（文件模块）

以下均为本程序的可选文件：
* cpu.so（CPU模块）
* version.so（系统模块）
* disk.so（硬盘模块）
* markdown.so（自述模块）
* style.css（HTML样式表）
* README.md（自述文件）
